CSS Statements

CSS statements are made up of four components:

  1. Declaration: the statement as a whole with curly braces.
  2. Selector: the name of the part you want to style.
  3. Property: properties to edit (e.g. width, height, background-color, list-style, etc.).
  4. Value: values that we assign (e.g. numbers, colors, urls, etc.)

CSS IDs

An ID is a UNIQUE identifier

ID Notes

  • In CSS an id starts with an octothrope (pound, hashtag, number sign, etc.) i.e. #
  • Example uses for IDs: targeting a specific header, or an about section.
  • Each ID should only be used once per page.

    • This is important with Javascript later on as well.
  • One-off styling

IDs versus Classes

IDs
